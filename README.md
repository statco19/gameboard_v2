# gameboard_v2
## 프로젝트 주제
League of Legends(LOL) API를 이용한 게임보드 제작

## 프로젝트 개요
LOL API로부터 LOL 솔로랭크 대전기록 데이터를 가져와 해당 패치의 승률, 밴률, 픽률을 구하고 각종 지표를 시각화하여 한눈에 메타를 살펴볼 수 있는 대시보드 제작하기

## 프로젝트 기술 스택
- AWS (AWC ECS, AWS EMR, AWS S3)
- Python (Python, Pandas, Pyspark)
- Spark (Spark SQL, Spark ML)
- ElasticSearch (ElasticSearch, Kibana)
- Javascript
